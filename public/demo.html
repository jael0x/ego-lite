ego-browser
Real Chrome, not a fake browser

Snapshot. Act. Re-snapshot.

A browser automation CLI for AI agents, built around semantic page snapshots, short-lived refs, and reusable site learnings.

1. ObserveRead the page with snapshot()
2. ActClick or fill using @eN refs
3. ReusePromote reliable flows into site tools
Quick start
$ego-browser <<'JS'
await newTab("https://example.com")
await waitForLoad()
const snap = await snapshot({ includeActionMarks: true, includeStableLocator: true })
print(snap.content)
await click("@e1")
✓ Re-snapshot after the page changes
@e1 short-lived ref from the latest snapshot
loc=... stable locator for reuse
siteSkills() load existing site learnings first
